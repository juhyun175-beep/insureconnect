# 📈 성장 포커스 루틴 (전산·렌트카 유입·매출전환 극대화)

앞으로 모든 개발은 **보험사 전산 유입 + 매출전환**(및 **리스/렌트카**) 극대화를 기본 렌즈로 한다.
사용자가 매번 지시하지 않아도 작업자(AI)가 이 체크리스트를 **자동 적용**한다. (보안 순찰 SECURITY.md 와 동일한 상시 루틴)

## 적용 대상(우선)
- 보험사 전산·청구: `/company`, `/company/[slug]`, `/ga`, 청구서류 허브
- 리스/렌트카(수익/제휴): `#page-rental`, 견적/제휴 도입부

## 모든 기능에 자동 적용할 5축
1. **유입(Acquisition)** — 검색 SEO(롱테일·JSON-LD) / 공유 OG·카톡 직접전송 / QR·UTM / 내부링크. *새 콘텐츠가 검색·공유로 들어올 길이 있는가?*
2. **전환(Conversion)** — 명확한 다음 행동 CTA(회원가입·견적문의·공유·구독). *전산/렌트카 페이지에 CTA가 있는가?*
3. **측정(Measurement)** — 진입·CTA에 추적 이벤트(`/api/track/hit`·`card-click`·`link-click`) 부착. *전환을 셀 수 있는가?*
4. **재방문(Retention)** — 즐겨찾기·웹푸시·카톡 알림·PWA로 직접 재유입 고착.
5. **수익(Revenue)** — 렌트카 견적/제휴 등 매출 연결 지점 노출·추적.

## 워크플로
기능 구현 → 위 5축 점검(빠진 축 보완) → 보안 순찰(`SECURITY.md`) → 배포.
