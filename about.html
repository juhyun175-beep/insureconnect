InsureConnect

보험설계사를 위한 통합 정보 허브.
각 보험사 전산, 청구서류, 소식지, 실무도구를 한 곳에서 빠르게 만나세요.

🎯 우리가 해결하는 문제

현장에서 보험설계사는 하루에도 수십 번씩 여러 보험사 전산을 오가며 작업합니다. 보험사마다 로그인 페이지, 청구서류 위치, 소식지 다운로드 경로가 다 다르고, 매번 검색해야 하는 번거로움이 컸습니다.

InsureConnect는 이러한 "흩어져 있는 보험사 자료를 한 페이지에서 모두 접근"할 수 있도록 만든 무료 도구입니다.

🛠 주요 기능

🖥
보험사 전산 모음
생명·손해보험사 32곳의 전산 사이트와 연락처 정보를 카드 한 장으로 정리
💳
수납 전산
카드 수납이 가능한 보험사 별도 정리로 결제 처리 시간 절감
📰
보험사 소식지
생명·손해보험사 월간 소식지 / 공시자료 바로가기
📋
청구서류
보험사별 청구서식 다운로드 / 양식 안내 자료
🛠
실무도구
실손계산기, 수술명 검색, KCD 질병코드, 할인할증 조회 등 12종
📚
보험지식
자체 제작 보험 상식 / 청구 노하우 / 실무 팁 콘텐츠
🗞
인슈어커넥트 뉴스
설계사들이 알아야 할 업계 동향을 카드뉴스 형태로 큐레이션
💼
채용공고
전국 GA / 보험사 채용 정보 통합 조회

👥 누가 사용하나요?

  • 현직 보험설계사 (생명·손해 GA, 전속, 법인대리점 소속 모두)
  • 설계사 매니저, 영업관리자
  • 보험 청구 / 보상 업무 종사자
  • 보험 영업을 시작하려는 신입 / 예비 설계사

💚 무료 서비스 운영 원칙

회원가입 없음

이메일이나 개인정보 입력 없이 누구나 즉시 이용 가능합니다.

광고로 운영

서비스 운영 비용은 Google AdSense 광고로 충당합니다. 광고가 다소 표시될 수 있는 점 양해 부탁드립니다.

중립 정보 제공

특정 보험사나 상품을 추천하지 않습니다. 모든 보험사 정보를 동등하게 게재합니다.

커뮤니티 친화

잘못된 정보나 추가했으면 하는 자료가 있다면 카카오톡 오픈채팅으로 언제든 연락 주세요. 함께 만들어가는 도구입니다.

🔒 안전성 / 개인정보

  • 회원가입이 없으므로 이용자의 이름·연락처·계정 정보는 수집·저장하지 않습니다.
  • 접속 통계용 익명 로그(IP 일부, 방문 일시, 클릭한 메뉴)만 1년간 보관 후 파기합니다.
  • 자세한 사항은 개인정보처리방침을 참고해 주세요.

📜 콘텐츠 출처

  • 자체 제작: 보험지식 포스트, 인슈어커넥트 뉴스(카드뉴스), 실무도구 정리
  • 외부 인용: 보험사 청구서류·소식지는 각 보험사 공식 자료를 링크 형태로 안내 (저작권은 해당 보험사)
  • 회사 로고: 생명보험협회(KLIA) / 손해보험협회(KNIA) 공식 자료실 이미지 사용
