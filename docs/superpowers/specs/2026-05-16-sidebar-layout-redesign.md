# 사이드바 레이아웃 개편 설계 문서

**날짜:** 2026-05-16  
**참고 사이트:** https://bohumschool-archive.pages.dev/desk

---

## 배경

현재 InsureConnect는 상단에 히어로 영역(로고 + 커뮤니티 카드 + 광고문의 카드)이 있고, 좌측 사이드바는 메뉴 항목이 최소화되어 있다. 커뮤니티 참여 버튼이 히어로 영역에 가로 배치되어 있어 공간 낭비가 크다.

참고 사이트처럼 사이드바에 메뉴 + 커뮤니티 + 부가 기능을 모두 통합하고, 히어로 영역을 제거해 콘텐츠 영역을 최대화한다.

---

## 목표

1. 히어로 영역(`.hero`, `.comm-section`, `.contact-card`) 완전 제거
2. 좌측 사이드바에 모든 네비게이션 + 커뮤니티 + 부가 기능 통합
3. 모바일: 기존 햄버거 슬라이드인 방식 유지

---

## 레이아웃 구조

### 데스크탑

```
┌──────────────────────┬──────────────────────────────┐
│  사이드바 (220px 고정) │  콘텐츠 영역 (나머지 전체)      │
│  (풀 높이, 스크롤)     │  (변화 없음)                   │
└──────────────────────┴──────────────────────────────┘
```

### 모바일

- 상단 헤더바(햄버거 + 로고) 유지
- 햄버거 클릭 시 사이드바 좌측에서 슬라이드인
- 외부 클릭 시 닫힘

## 사이드바 내부 구성 (위→아래)

### 1. 헤더 영역
- `logo-banner.png` 이미지 (현재와 동일)
- "메뉴 숨기기" 버튼 (현재 없음 → 추가, 데스크탑 전용)

### 2. 메뉴 탭 영역
기존 `.menu-tab` 버튼들 전체 유지:
- 🏠 홈
- 🗞 카드뉴스
- 🖥 보험사 전산
- 💳 수납 전산
- 📄 소식지
- 🛠 실무도구
- 👥 커뮤니티

### 3. 커뮤니티 참여 버튼 (히어로에서 이동)
히어로의 `.comm-card`를 사이드바용 컴팩트 버튼으로 변환:
- **카카오 CREW 300** — 카카오 오픈채팅 링크
- **매니저링크** — 매니저 오픈채팅 링크
- GM커넥트는 제외

스타일: 아이콘 + 이름 + `›` 화살표 한 줄 구성, 호버 시 배경 하이라이트

### 4. 피드백 / 공유 영역
- 📝 수정요청 / 제안하기 링크
- 📤 동료에게 공유하기 버튼 (현재 기능 그대로)

### 5. 방문자 통계
- 오늘 방문자 / 누적 방문자 표시 (정적 UI, 실제 카운터 미연동)

### 6. 테마 토글
- [라이트] [시스템] [다크] 3단 토글 버튼
- `localStorage` 기반 `data-theme` 속성 전환
- 다크 테마 CSS 변수는 기본 구현 (추후 상세 스타일 보완 가능)

### 7. 푸터
- 서비스명 텍스트
- 개인정보처리방침 링크

## 제거 대상

| 요소 | 설명 |
|------|------|
| `.hero` | 상단 히어로 섹션 전체 |
| `.hero-inner`, `.hero-main-row` | 히어로 내부 구조 |
| `.comm-section`, `.comm-card` | 히어로 내 커뮤니티 카드 |
| `.contact-card` | 광고/협업 문의 카드 |
| `.hero-divider`, `.hero-badge` | 히어로 장식 요소 |
| `.hero-strip` | 히어로 하단 스트립 |
| 관련 CSS 전체 | 위 요소들의 스타일 규칙 |

## 비변경 사항

- 콘텐츠 영역 (`#page-home`, `#page-jeonsan` 등) — 변화 없음
- 사이드바 너비 (`--side-w` CSS 변수) — 현재값 유지
- 모바일 햄버거 슬라이드인 동작 — 현재 로직 유지
- 기존 메뉴 탭 페이지 이동 로직 — 변화 없음
